# 🚀 DecodeLabs Internship - Data Analytics

## 📌 Overview
This repository contains my work for the DecodeLabs Data Analytics Internship.  
The internship focuses on building strong foundations in data cleaning, analysis, and real-world problem solving.

---

## 🗓️ Week 1: Data Cleaning & Preparation 🧹

### 📊 Objective:
Clean and prepare raw e-commerce dataset for analysis.

### 🛠️ Tasks Performed:
- Loaded dataset using pandas
- Identified missing values
- Handled missing data in `CouponCode` column
- Verified and removed duplicate records
- Standardized categorical data
- Ensured correct data types

### 📈 Key Insights:
- Most used payment method: Online (258 orders)
- Top referral source: Instagram (259 users)

### 📂 Files:
- `data.xlsx` → Raw dataset  
- `clean_data.py` → Data cleaning script  
- `cleaned_data.csv` → Final cleaned dataset

## 🗓️ Week 2: Exploratory Data Analysis (EDA) 📊

### 📌 Objective
To analyze the cleaned e-commerce dataset and extract meaningful insights that can support business decision-making.

---

### 🛠️ Tasks Performed

#### 🔹 1. Revenue Analysis
- Calculated total revenue generated from all orders
- Computed average order value to understand customer spending behavior

#### 🔹 2. Payment Method Analysis
- Analyzed distribution of different payment methods
- Identified the most preferred payment options among customers

#### 🔹 3. Product Analysis
- Identified top-selling products based on frequency
- Evaluated product demand trends

#### 🔹 4. Customer Acquisition Analysis
- Analyzed referral sources (Instagram, Email, Google, etc.)
- Determined which channels bring the most customers

#### 🔹 5. Time-Based Analysis
- Extracted month from order date
- Analyzed number of orders per month
- Identified seasonal trends in sales

#### 🔹 6. Data Visualization
- Created bar charts for:
  - Payment method distribution
  - Referral source distribution
- Used visual representation to simplify insights
